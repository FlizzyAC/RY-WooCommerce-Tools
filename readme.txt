== Description ==

Let your WooCommerce store support ECPay payment gateway and ECPay shipping.

Let your WooCommerce store more with Taiwan's habits.

== Installation ==

= Minimum Requirements =

* php 5.5.0+
* WordPress 5.0+
* WooCommerce 3.0+
