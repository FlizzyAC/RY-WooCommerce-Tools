Accept your WooCommerce store payment with ECPay / NewebPay and shipping with ECPay.

== Description ==

Let your WooCommerce store support ECPay payment gateway, ECPay shipping and NewebPay payment gateway.

Let your WooCommerce store more with Taiwan's habits.

== Installation ==

= Minimum Requirements =

* PHP 5.6.20+
* WordPress 5.0+
* WooCommerce 3.0+
